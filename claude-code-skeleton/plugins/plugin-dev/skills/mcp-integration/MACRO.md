# Skill: `mcp-integration`

**Role:** Guides **Model Context Protocol** integration: server flavors, authentication concerns, and how Claude Code discovers tools from MCP config.

**Contents:** `SKILL.md` is the entry point with triggers and a decision tree for MCP adoption. `references/` provides detailed write-ups on tool invocation, server types, and authentication patterns. `examples/` contains JSON configuration templates for different MCP transport styles (`stdio`, `sse`, `http`).
