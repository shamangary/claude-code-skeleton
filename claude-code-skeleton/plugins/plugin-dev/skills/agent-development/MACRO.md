# Skill: `agent-development`

**Role:** Authoring **agents**: structure, tool allow-lists, validation, worked TS/Python SDK examples, and helper scripts. Activates on triggers related to composing multi-agent workflows.

**Contents:** `SKILL.md` is the entry point for triggers and essential patterns. `references/` provides deep guidance on robust agent prompts, system prompt design, and triggering examples. `examples/` contains language-specific agent samples (agent-creation prompt, complete examples). `scripts/validate-agent.sh` provides authoring automation for validating agent markdown and packaging.
