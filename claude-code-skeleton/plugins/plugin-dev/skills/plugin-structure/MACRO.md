# Skill: `plugin-structure`

**Role:** Canonical **layout** for Claude Code plugins: folders, manifest fields, minimal vs advanced examples, marketplace packaging notes.

**Contents:** `SKILL.md` (with `README.md` as a stub index) is the entry point for quick orientation. `references/` provides deep dives into manifest keys and component composition patterns. `examples/` contains worked plugin shapes (minimal → standard → advanced) that authors can compare side-by-side.
