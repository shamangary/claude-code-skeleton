# Plugin: `frontend-design`

**Role:** Biases **frontend work** toward **distinctive, production-grade** UI (typography, motion, visual identity) and away from generic "AI slop" aesthetics—via an auto-invoked **skill**. The `plugin.json` declares the plugin for Claude Code loader / marketplace.

**Audience:** Frontend implementers using Claude Code.

**Skill (`skills/frontend-design/`):** `SKILL.md` encodes trigger phrases, design principles, and constraints the model should follow when doing UI/UX implementation work.
